#  Lab 3: Router Basic Configuration

> **Tools Used:** Cisco Packet Tracer / GNS3  
> **Focus:** Connecting two networks using a router, configuring interfaces, and testing connectivity  

---

## 🎯 Objectives

- Understand **router components** and **IOS boot sequence**
    
- Configure **router interfaces** and **IP addresses**
    
- Enable connectivity between two separate networks
    
- Test connectivity using `ping`
    
- Save and verify router configuration
    

---

## 🔹 1️⃣ Quick Review

- **Router Components:** ROM, RAM, Flash, NVRAM
    
- **IOS:** Operating System (stored in Flash)
    
- **Startup-config:** Stored in NVRAM
    
- **Boot Sequence:** Power On → IOS → Startup-config
    

---

## 🔹 2️⃣ Lab Concept

- Two separate networks: each with a **Switch + PCs**
    
- Test connectivity inside each network (Ping ✅)
    
- Test connectivity between networks (Ping ❌ initially)
    
- Add a **Router** to connect networks
    

---

## 🔹 3️⃣ Network Setup

|Network|IP Range|Devices|Notes|
|---|---|---|---|
|Network 1|192.168.1.0/24|PC1: 192.168.1.2  <br>PC2: 192.168.1.3|Ping works within network|
|Network 2|192.168.2.0/24|PC3: 192.168.2.2  <br>PC4: 192.168.2.3|Ping works within network|

- Ping between networks initially fails ❌
    

---

## 🔹 4️⃣ Adding the Router

- Router: Cisco 1941 (example)
    
- Connections:
    
    - Switch1 → Router (Gig0/0)
        
    - Switch2 → Router (Gig0/1)
        
- **Note:** Interfaces are shutdown by default → must run `no shutdown`
    

---

## 🔹 5️⃣ Router Modes

|Mode|Prompt|Description|
|---|---|---|
|User EXEC|`Router>`|Basic commands (`ping`)|
|Privileged EXEC|`Router#`|`enable`|
|Global Configuration|`Router(config)#`|`configure terminal`|
|Interface Configuration|`Router(config-if)#`|`interface ...`|

> **Tip:** Use `do` to run commands from any mode:

```lua
Router(config)# do show clock  
Router(config)# do clock set 08:06:10 10 May 2025
```
---

## 🔹 6️⃣ Router Configuration Steps

1. Enter CLI:
    

```lua
Router> enable  
Router# configure terminal
```

2. **Change router hostname** (optional):
    
```lua
Router(config)# hostname R1  
R1(config)#
```

3. **Configure Gig0/0 (Network 1)**:
    
```lua
R1(config)# interface gig0/0  
R1(config-if)# ip address 192.168.1.1 255.255.255.0  
R1(config-if)# no shutdown  
R1(config-if)# exit
```
3. **Configure Gig0/1 (Network 2)**:
    
```lua
R1(config)# interface gig0/1  
R1(config-if)# ip address 192.168.2.1 255.255.255.0  
R1(config-if)# no shutdown  
R1(config-if)# exit
```

5. **Verify interface status**:
    
```lua
R1# show ip interface brief
```

6. **Check routing table**:
    
```lua
R1# show ip route
```

> Single router → no routing protocol required

7. **Set Default Gateway for PCs**:
    

- Network 1 → 192.168.1.1
    
- Network 2 → 192.168.2.1
    

---

## 🔹 7️⃣ Save Configuration

```lua
R1# wr
```

or

```lua
R1# copy running-config startup-config
```

---

## 🔹 8️⃣ Test Connectivity

- Ping between networks → should succeed ✅
    
- Example: PC1 (192.168.1.2) → `ping 192.168.2.2`
    

---

## 🔹 9️⃣ Important Notes

- Always `no shutdown` for new interfaces
    
- Use `TAB` for auto-completion
    
- On first boot, choose `No` for Setup Dialog
    
- Console access possible via **Putty**
