> **Tools Used:** Cisco Packet Tracer / GNS3  
> **Focus:** Configuring RIP (Dynamic Routing) to enable inter-network connectivity

---

## 🎯 Objectives

- Understand **RIP** (Routing Information Protocol)
    
- Learn **RIPv1 vs RIPv2** differences
    
- Configure **RIP on Cisco routers**
    
- Test connectivity between distant networks using `ping` and `tracert`
    

---

## 🔹 1️⃣ RIP Overview

- **RIP** = Routing Information Protocol
    
- One of the oldest **Dynamic Routing Protocols**
    
- Uses **Distance Vector Routing**
    
- **Metric:** Hop Count (number of routers to reach the destination)
    

---

### 🔹 Key Features of RIP

|Feature|Details|
|---|---|
|Metric|Hop Count (Max = 15 → >15 unreachable)|
|Routing Updates|Every 30 seconds to neighbors|
|Transport Protocol|UDP, Port 520|
|Versions|RIPv1 (Classful, no subnet mask), RIPv2 (Classless, supports VLSM, multicast 224.0.0.9)|

---

### 🔹 Advantages & Disadvantages

**Advantages:**

- Simple to configure
    
- Suitable for small networks
    

**Disadvantages:**

- Limited scalability (15 hops max)
    
- Slow convergence
    
- Uses only hop count (ignores bandwidth/delay)
    

---

### 🔹 RIP Mechanism

1. Router sends its routing table to neighbors every 30 sec
    
2. Each router adds +1 hop to received routes
    
3. Updates its routing table with best routes
    
4. Process repeats continuously
    

---

### 🔹 RIP Loop Prevention

- **Split Horizon:** don’t send route back on same interface
    
- **Route Poisoning:** mark failed route as hop count = 16 (unreachable)
    
- **Hold Down Timer:** ignore incorrect updates for a period

## 🔹 2️⃣ Cisco RIP Commands

### Enable RIP

```lua
Router> enable  
Router# configure terminal  
Router(config)# router rip  
Router(config-router)# version 2  
Router(config-router)# network 192.168.1.0  
Router(config-router)# network 10.0.0.0
```
### Disable Auto-Summarization

```lua
Router(config-router)# no auto-summary
```

### Verification

```lua
Router# show ip route  
Router# show ip protocols
```

---

### 🔹 RIPv1 vs RIPv2

|Feature|RIPv1|RIPv2|
|---|---|---|
|Classful|Yes|No|
|VLSM|No|Yes|
|Subnet Mask|Not sent|Sent|
|Broadcast/Multicast|Broadcast|Multicast (224.0.0.9)|
|Authentication|No|Yes|

## 🔹 4️⃣ Configure Router Interfaces

### R1

```lua
enable  
configure terminal  
interface g0/0  
 ip address 192.168.1.1 255.255.255.0  
 no shutdown  
exit  
interface s0/0  
 ip address 10.0.0.1 255.255.255.252  
 no shutdown  
exit
```

### R2

```lua
enable  
configure terminal  
interface s0/0  
 ip address 10.0.0.2 255.255.255.252  
 no shutdown  
exit  
interface s0/1  
 ip address 10.0.0.5 255.255.255.252  
 no shutdown  
exit
```

### R3

```lua
enable  
configure terminal  
interface s0/0  
 ip address 10.0.0.6 255.255.255.252  
 no shutdown  
exit  
interface g0/0  
 ip address 192.168.3.1 255.255.255.0  
 no shutdown  
exit
```

---

## 🔹 5️⃣ Configure RIP

### R1

```lua
router rip  
 version 2  
 no auto-summary  
 network 192.168.1.0  
 network 10.0.0.0
```
### R2

```lua
router rip  
 version 2  
 no auto-summary  
 network 10.0.0.0
```
### R3

```lua
router rip  
 version 2  
 no auto-summary  
 network 10.0.0.0  
 network 192.168.3.0
```
