#  Lab 2: Subnetting Practical Example

> **Tools Used:** Cisco Packet Tracer / GNS3 (Optional)  
> **Focus:** Understanding subnetting, calculating subnet masks, host addresses, and broadcast addresses

---

## 🎯 Objectives

- Practice subnetting a given network
    
- Calculate new **Subnet Mask** after borrowing bits
    
- Determine **number of hosts per subnet**
    
- Create a **Subnetting Table** (Network ID, First Host, Last Host, Broadcast)
    

---

## 🔹 1. Problem Statement

Given network: `192.168.10.0/24`  
Task: Divide it into **8 subnets** and find:

- New Subnet Mask
    
- Number of usable hosts per subnet
    
- Subnet distribution table
    

---

## 🔹 2. Step-by-Step Solution

### Step 1️⃣ Determine number of borrowed bits (n)

- Required subnets: 8
    
- Formula: `2^n = number of subnets`
    
- 2^3 = 8 → borrow **3 bits**
    

---

### Step 2️⃣ New Subnet Mask

- Original mask: `/24 = 255.255.255.0`
    
- Borrow 3 bits → `/27`
    
- Decimal calculation for last octet:
    
    - 11100000 = 128 + 64 + 32 = **224**
        

✅ New Subnet Mask: `255.255.255.224` (/27)

---

### Step 3️⃣ Number of Hosts per Subnet

- Remaining bits for hosts: 5
    
- Formula: `(2^m) - 2`
    
- (2^5) - 2 = 30 hosts per subnet
    

---

### Step 4️⃣ Block Size

- Block = 256 - interesting octet
    
- 256 - 224 = **32**
    
- Each subnet starts **32 addresses after the previous one**
    

---

---

### Step 6️⃣ Quick Formula to Calculate Network IDs

1. Calculate **Block Size** = `256 - interesting octet`
    
2. Start from **first Network ID** (base network)
    
3. Add Block Size sequentially
    

Example:

|Subnet #|Network ID|
|---|---|
|1|192.168.10.0|
|2|192.168.10.32|
|3|192.168.10.64|
|4|192.168.10.96|
|5|192.168.10.128|
|6|192.168.10.160|
|7|192.168.10.192|
|8|192.168.10.224|

> **Note:** Last subnet = 224 → Broadcast = 255

---

### ✅ Final Results

- **New Subnet Mask:** `/27 = 255.255.255.224`
    
- **Number of Subnets:** 8
    
- **Number of usable hosts per subnet:** 30
